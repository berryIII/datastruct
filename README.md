基础数据与算法的总结，使用C实现，可以用于备考或面试时的回顾参考。

## 线性数据结构
```
├───线性表(sequence_table)
│   ├───顺序存储
│   │   ├───静态
│   │   └───动态扩容
│   └───链式存储
│       ├───普通单链表
│       └───带头结点链表
│           ├───单向链表
│           │───双向链表
│           └───循环链表
│
├───栈(stack)
│   ├───顺序存储
│   ├───链式存储
│   └───经典问题
│       ├───出栈的可能组合——卡塔数
│       └───中缀转后缀与后缀表达式计算
│
└───队列(queue)
    ├───顺序存储
    │   ├───顺序队列
    │   └───循环队列
    └───链式存储        
```

## 树形结构
```
├───二叉树(binary tree)
│   ├───顺序存储
│   ├───链式存储
│   │     ├───递归与非递归前中后序遍历
│   │     └───恢复二叉树
│   │
│   ├───完全二叉树
│   │
│   ├───二叉树中的计算公式
│   │
│   ├───线索二叉树
│   │
│   ├───二叉搜索树(BST)
│   │     └───增删查，ASL
│   │
│   └───哈夫曼树
│
└───堆(heap) // TODO
```

## 图
```
├───图(无重边)
│   ├───基本概念
│   │   ├───顶点/边/度/完全图
│   │   ├───连通图/强连通图
│   │   └───子图
│   │       ├───连通分量/强连通分量
│   │       └───生成树/最小生成树
│   │ 
│   ├───存储
│   │   ├───邻接矩阵
│   │   └───邻接表
│   │ 
│   ├───加权图
│   │    ├───AOV
│   │    │   └───拓扑排序
│   │    └───AOE
│   │        └───关键路径问题
│   └───经典问题
        ├───连通检测/连通分量计算
        ├───最小生成树
        ├───最短路径
        └───拓扑排序
```
